**Lab 1: Python Refresher**

Definition: An introductory lab focused on reinforcing core Python programming and text-handling fundamentals.
Objective: Apply Python basics to file handling, BMI calculations, string operations, and text processing tasks.

**Lab 2: Parsing and Text Cleaning**

Definition: A lab centered on parsing and cleaning noisy text data extracted from real-world sources.
Objective: Remove noisy text, clean user comments, replace slang, and evaluate data-cleaning efficiency using character-based metrics

**Lab 3: Natural Language Processing with NLTK**

Definition: A foundational NLP lab introducing text preprocessing and linguistic analysis using the NLTK library.
Objective: Perform tokenization, stop-word removal, stemming, lemmatization, POS tagging, NER, and frequency analysis on text data.

**Lab 4: Natural Language Processing with spaCy**

Definition: A practical lab exploring modern NLP workflows using spaCyâ€™s language processing pipeline.
Objective: Apply tokenization, lemmatization, sentence segmentation, stop-word removal, POS tagging, and named entity recognition in an integrated workflow.

**Lab 5: Text Processing Using TF-IDF**

Definition: A lab focused on feature extraction and term importance analysis using TF-IDF modeling.
Objective: Compute and interpret TF-IDF scores, tune vectorizer parameters, and identify important terms across text datasets. 

**Lab 6: Text Analysis Using TF-IDF, Bag of Words, and N-Grams**

Definition: A lab focused on representing text data using multiple feature extraction techniques to understand term importance and context.
Objective: Build and compare Bag of Words, TF-IDF, and N-Gram models to analyze word frequency, relevance, and relationships in text data.

**Lab 7: Text Classification Using Multiple Machine Learning Algorithms**

Definition: A supervised learning lab that applies and compares multiple classification algorithms on large-scale text data.
Objective: Preprocess text, extract TF-IDF features, train Naive Bayes, Logistic Regression, SVM, and KNN models, and evaluate their performance.

**Lab 8: Text Clustering with Multiple Algorithms**

Definition: An unsupervised learning lab that explores clustering techniques for discovering patterns in text data.
Objective: Apply K-Means, Hierarchical Clustering, and DBSCAN on TF-IDF vectors and evaluate cluster quality using silhouette-based metrics.

**Lab 9: Enhancing a Movie Recommender System**

Definition: A systems-focused lab that improves recommendation accuracy by integrating multiple recommendation approaches.
Objective: Build and evaluate a hybrid recommender system combining content-based and collaborative filtering using the MovieLens dataset.

**Lab 10: Sentiment Analysis Using VADER and Hugging Face Transformers**

Definition: A comparative NLP lab examining lexicon-based and transformer-based sentiment analysis techniques.
Objective: Perform sentiment analysis using VADER and Hugging Face models, compare predictions, and evaluate performance using standard metrics.
